## Scripts Description

### Script 1
Displays system info like kernel, user, uptime.

Shows disk usage and permissions of directories.

### Script 4
Analyzes log file and counts errors.

### Script 5
Generates open-source manifesto using user input.

## How to Run

Make scripts executable:
chmod +x *.sh

Run scripts:
./script1_system_identity.sh  
./script2_package_inspector.sh  
./script3_disk_auditor.sh  
./script4_log_analyzer.sh /var/log/syslog  
./script5_manifesto.sh
